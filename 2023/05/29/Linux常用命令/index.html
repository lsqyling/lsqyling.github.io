Linux常用命令

Posted on 2023-05-29 Edited on 2023-08-26

前言：熟练掌握常用的Linux命令，可以提高生产力，研究问题的能力。常用命令包括：文件操作，系统查看，GCC命令，GDB调试等。这些都在系统开发中起到很重要的作用。

File

  • ls /etc/a*.conf (匹配查找文件)
  • grep “#define container_of” -R .(当前目录递归查找)
  • diff /etc/service tmp (比较文件的差异)
  • cat tmp
  • tail -n 10 out.log (-f 动态模式)
  • chmod a+x a.out (更改文件的权限 常用权限664可读可写可执行)
  • chown root a.out（将 a.out文件的所有者更改为 root 用户）
  • ./a.out -> sudo !! (当执行a.out 没有权限时可以使用 sudo !! 获得权限执行)
  • chmod u+s mysu

    设置文件或目录的 setuid 位。当一个带有 setuid 位的文件或目录被执行时，它会以该文件或目录的所有者的权限而不是执行它的用户的权限来运行。这对于需要以提升的权限运行但不能作为 root 运行的程序非常有用。如果您想允许用户运行需要 root 权限的程序而不给他们访问 root 帐户，可以在程序文件上设置 setuid 位。

  • du abc.c/directory (分析文件/目录占用磁盘空间大小 默认是以k字节为单位)
  • primer0 | wc -l (统计primer0执行输出结果的行数)
  • cat file | wc -l (统计file文件内容的行数)
  • file a.out (查看文件的类型)
  • ls -lh (较好的方式显示文件的大小)

System

  • ulimit -a (查看当前用户的资源限制，如最大打开文件数、最大进程数等等)
  • strace ./a.out (看可执行文件的系统调用是如何发生的)
  • time ./a.out (查看调用时间)
  • export (显示所有的环境变量)
  • ps axf（显示进程,及进程下子进程）
  • ps axm（显示进程不显示递进关系）
  • ps axj（显示进程的状态信息，包括进程的 PID、PPID、CPU 占用率、内存占用率等等）
  • ps ax -L（显示进程和线程）
  • pstree (打印系统进程树)
  • pmap p_id（查看程序的进程空间的分布）
  • kill -l（list all signal）
  • kill pid（杀死进程）
  • ipcs (查看系统进程间通信的信息)

Network

  • source <(curl -sL https://multi.netlify.app/v2ray.sh) --zh（安装v2ray）
  • systemctl stop firewalld.service （关闭防火墙）
  • systemctl disable firewalld.service
  • sudo ufw disable (Debian禁用ufw防火墙)
  • ip addr (查看网络环境)
  • ip add sh(查看网卡信息)
  • ip ro sh(查看ip路由表的命令)
  • ip ro add default via 192.168.1.125（添加默认路由）
  • ipcs（显示系统IPC）
  • netstat -anu(local pc udp info)
  • netstat -ant(local pc tcp info)

GCC

  • gcc -E hello.c
  • gcc -E hello.c > hello.i
  • gcc -S hello.i
  • gcc -c hello.s
  • gcc hello.o -o hello
  • gcc hello.c
  • gcc hello.c -o myhello
  • make hello
  • objdump -d a.out （反汇编 汇编码-d）
  • strace -f gcc a.c (追踪gcc a.c 的过程，包括gcc 启动的其他程序) strace -f gcc a.c 2>&1 | grep execve
  • strip test_sqlist (移除调试信息)

GDB

  • gdb a.out (载入程序)
  • run/r (开始运行程序)
  • starti (运行第一条指令)
  • layout asm (查看汇编代码)
  • si (单步查看->汇编单步)
  • info registers (查看寄存器代码)
  • bt (打印系统调用栈)
  • info inferiors（打印可执行程序的进程号）
  • print/p $rdx (打印寄存器rdx的值)
  • target record-full (开始记录)
  • watch $rax（观察寄存器rax的值）
  • watch *0x7c00 (在地址7c00处设置监视点)
  • rsi (单步回退查看->汇编单步)
  • layout src(调试源代码)
  • set scheduler-locking on (可以使其他处理器（线程)不要干扰）
  • thread 1（switching to thread 1）
