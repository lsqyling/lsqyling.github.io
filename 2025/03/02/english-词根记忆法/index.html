英语词根记忆法

Posted on 2025-03-02

简介：英语词根记忆法是一种通过学习和理解单词的基本构成部分——词根、前缀和后缀来增强词汇量和理解能力的学习策略。这种方法基于许多英语单词都是由拉丁语和希腊语的词根演变而来的事实，因此掌握这些基本组成部分可以帮助学习者更轻松地推测出大量单词的意思。

词缀词根记忆法

后缀-cess，-cede，-ceed，-gress，-grad，-gred

  • 均表示走=to go

前缀sub-，sup-，suc-，sus-的功能

  • 下，向下: subway
  • 从下往上：submit
  • 亚于，次于: suburban
  • 后，后面: subsequent

com-，con-，col-，cor-，co-

  • =with/together，共同、一起、全部
  • intensive，加强语气

anc-，ant- 反

  • 以前、前、前面、老
  • 高

abs-

  • =away

ac-

  • =to

inter-

  • xx之间

se-

  • =apart分开、分离

di-，dis-

  • =apart分开

trans-

  • 转，穿，传

词根 “-van，-ven，-vent” = come、go 都表示 来或是走来的含义

in- im- il- ir-

  • not:相反、否定
  • =in，进入
  • =en，使

pre- = 前、提前； pro- = forward

-able -ible -ile

  • adj:可以xxx的，能够xxx的
  • adj:容易xxx的
  • N(少数): vary —> variables 名词后缀 变量

re-

  • 反复、又、再
  • 返回
  • 相反

e- ex- es-

  • 向外、出来
  • 超出
  • ex- 还表示：前 for-example: ex-boyfriend

ad-

  • =to，去
  • 加强

词根-vent、-ven-van表示"来，走来
